# Coding Foundations

Dieses Repository dient als persönliche Wissensdatenbank und Sammlung für die Implementierung grundlegender Konzepte der Informatik und Programmierung.
Mein Ziel ist es, die Funktionsweise von Datenstrukturen und Algorithmen nicht nur theoretisch zu verstehen, sondern durch saubere, eigenständige Implementierungen zu festigen.
Um wirklich den Kern der Informatik zu verstehen habe ich mich deswegen für C entschieden.

## 🚀 Ziel des Projekts
Das Repository fungiert als Lernumgebung, um:
* Die Funktionsweise essenzieller **Datenstrukturen** zu begreifen.
* **Algorithmen** effizient zu implementieren.
* Sauberen, modularen und gut dokumentierten Code zu schreiben.
* Eine Referenz für zukünftige Projekte aufzubauen.
